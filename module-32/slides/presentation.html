Custom Classifiers for AI Data

Create AI-specific classification rules with regex patterns and ML-based matching.

# Define custom classification for AI training data
custom_classification = {
  "name": "AI_Training_Dataset",
  "classificationRuleType": "Custom",
  "classificationRules": [{
    "kind": "RegexClassificationRule",
    "pattern": r"training[_-]data[_-]\d{8}",
    "minimumPercentageMatch": 60
  }]
}

Tip: Combine regex classifiers with trainable ML classifiers for complex AI data patterns that simple patterns can't capture.
