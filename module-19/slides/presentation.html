What Is Function Calling?

The mechanism that turns a text generator into an active agent.

⚡ Key insight: The model never executes code. It outputs a JSON request; your application runs the function and returns the result.

Anatomy of a Tool Definition

{
  "type": "function",
  "function": {
    "name": "get_weather",           // Unique identifier
    "description": "Get current     // When to call
      weather for a city...",
    "parameters": {                // JSON Schema
      "type": "object",
      "properties": {
        "city": { "type": "string" },
        "units": { "enum": ["celsius", "fahrenheit"] }
      },
      "required": ["city"],
      "additionalProperties": false
    }
  }
}

Three pillars: name (what), description (when/why), parameters (how)

JSON Schema Best Practices

PracticeWhy It MattersExample
enumConstrains to valid values["celsius", "fahrenheit"]
patternValidates format"^\\d{4}-\\d{2}-\\d{2}$"
min/maxBounds numeric ranges"minimum": 15, "maximum": 480
requiredEnsures essential fields["city", "date"]
additionalProperties: falsePrevents hallucinated fieldsAlways set on root

💡 Keep schemas flat (max 2 levels). Deep nesting reduces model accuracy.

Pattern 2: Parallel Calling

Model requests multiple independent functions simultaneously.

User: "Weather in
3 cities?"
→
get_weather("Seattle") get_weather("Portland") get_weather("Vancouver")
→
Combined
Response
async def handle_parallel(tool_calls):
    tasks = [dispatch(call) for call in tool_calls]
    results = await asyncio.gather(*tasks)
    return results

Benefit: Reduces latency — all calls execute concurrently

The Function Router Pattern

class FunctionRouter:
    def __init__(self):
        self._registry: dict[str, Callable] = {}

    def register(self, name: str, func: Callable):
        self._registry[name] = func

    async def dispatch(self, tool_call) -> str:
        func = self._registry.get(tool_call.function.name)
        if not func:
            return json.dumps({"error": "Unknown function"})
        try:
            result = await func(**json.loads(tool_call.function.arguments))
            return json.dumps(result)
        except Exception as e:
            return json.dumps({"error": str(e)})

📌 Centralizes dispatch, error wrapping, and extensibility.

Retry & Circuit Breaker

Exponential Backoff

for attempt in range(retries):
  try:
    return await func()
  except TransientError:
    delay = base * (2 ** attempt)
    await sleep(delay)

Handles transient failures with increasing wait times + jitter.

Circuit Breaker

# States:
CLOSED  → healthy, allow calls
OPEN    → failing, block calls
HALF    → test with single call

# Triggers:
3+ failures → OPEN
timeout expires → HALF-OPEN
success → CLOSED

Prevents cascade failures when services are down.

Structured Error Responses

✅ Good

{
  "error": "service_unavailable",
  "message": "Weather API 503",
  "retry_after": 30,
  "fallback_data": {
    "last_known_temp": 18,
    "data_age_min": 45
  }

💡 Include fallback_data so the model can still provide an approximate answer.
