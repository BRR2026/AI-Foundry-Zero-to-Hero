Core Concept

What is Azure AI Foundry?

🏭

The AI App & Agent Factory

Azure AI Foundry is Microsoft's unified platform for building, evaluating, and deploying AI applications and agents — from prototype to production, all in one place.

🔄 Formerly

Azure AI Studio — rebranded and expanded in late 2024 as AI Foundry to reflect its factory-like, end-to-end workflow.

🌐 Portal

Accessible at ai.azure.com — a web-based workspace for teams to build AI solutions with no-code and pro-code tools.

🛠️ SDK & CLI

Full programmatic access via the Azure AI Foundry SDK (Python, .NET, JS) and the az ai CLI extension.

🏢 Enterprise-Grade

Built-in RBAC, networking, content safety, compliance certifications, and integration with Azure DevOps & GitHub.

Platform Pillars

Key Capabilities Overview

🤖

Models

Browse, compare, and fine-tune 1,800+ models from the Model Catalog across text, vision, speech, and embeddings.

🕵️

Agents

Build autonomous AI agents with tool-use, code execution, file search, and multi-step reasoning — powered by the Agent Service.

🎮

Playground

Interactive sandbox to test prompts, tweak parameters, compare models side-by-side, and iterate in real time — no code required.

📊

Evaluation

Automated and manual evaluation pipelines with built-in metrics: groundedness, relevance, coherence, fluency, and safety scores.

🚀

Deployment

One-click deploy to serverless endpoints, managed compute, or Kubernetes — with autoscaling, blue-green, and A/B testing support.

🛡️

Responsible AI

Content safety filters, jailbreak detection, prompt shields, PII redaction, and built-in fairness analysis to ship AI you can trust.

Vocabulary

Key Terms to Know

AI Foundry Hub
Top-level Azure resource; holds shared settings, connections, and policies for all child projects.
Project
An isolated workspace under a Hub where teams build, test, and deploy AI solutions.
Model Catalog
Marketplace of 1,800+ pre-trained models from OpenAI, Meta, Microsoft, and open-source providers.
Deployment
A model served as a REST endpoint — serverless, managed compute, or self-hosted.
Playground
Browser-based UI to send prompts, tune parameters, and compare model outputs interactively.
Agent Service
Managed runtime for building AI agents with tool-use, code execution, and file search.
RAG
Retrieval-Augmented Generation — grounding LLM responses with external data for accuracy.
Content Safety
Azure service that detects harmful content, jailbreaks, and prompt injection attacks.
Fine-Tuning
Training an existing model on your own data to improve performance on domain-specific tasks.
Evaluation
Automated scoring of model outputs on metrics like groundedness, coherence, and safety.
Managed Compute
Azure-hosted GPU infrastructure for model hosting; scales automatically with demand.
Prompt Engineering
The practice of crafting instructions (prompts) to guide LLM behavior and output quality.

Recap

Module 1 Summary

  • ✅ AI Foundry is Microsoft's unified platform — the "AI App & Agent Factory" for building, evaluating, and deploying AI solutions at enterprise scale.
  • ✅ The Model Catalog offers 1,800+ models from OpenAI, Meta, Microsoft, and the open-source community — deployable with one click.
